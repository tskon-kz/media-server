# Media Server

> [English version](README.md)

Самохостинг медиасервера: Jellyfin + qBittorrent, управление полностью через Telegram-бот. Один скрипт — и всё готово.

## Возможности

- **Telegram Mini App** — полноценный веб-интерфейс через кнопку меню в чате (список торрентов, поиск, добавление, настройки)
- Добавляй торренты через Mini App: вставь magnet-ссылку или загрузи `.torrent`-файл
- **Поиск раздач** через Mini App — ищет по всем индексаторам Jackett, выбери результат и добавь
- Управление паролем администратора Jackett прямо из бота (сменить, убрать)
- Уведомления о завершении загрузки с автоматическим сканированием Jellyfin
- **Два режима хардлинков**: оригинальная структура (по умолчанию) или автоматическое переименование по стандарту Jellyfin — переключается для каждого торрента или глобально
- Управление категориями-библиотеками (Фильмы, Сериалы + добавление своих)
- Встроенная система обновлений через бота

## Установка

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/tskon-kz/media-server/main/install.sh)
```
